To make the entry into the framework as easy as possible we created a template project. The template project contains all necessary dependencies for a framework application. Just click here TODO, clone the repository and follow the given instructions.

After cloning execute the following instructions inside the template project's root folder in the terminal.

Get started with the template project

npm install
npm run dev

Then open localhost:3000 in your browser. That's it!

The template project also helps you to understand the following topics:

  • Authentication system (Provider and view)
  • Configuration of the header (logos etc)
  • Configuration of the navigation bar and the navigation bar tabs
  • Configuration of the content bar and the content bar tabs
  • Translation system

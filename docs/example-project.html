Starting the Example Project

Important hint: If you haven't added the ca file of IAV in npm yet, you will have to read the guide "Error: Self signed certificate in chain" in FAQ first.

The framework's repository contains an example project which is used for development purposes and to present a basic example for framework usage. If you want to start the example project, first you have to clone the framework's repository.

You can start the example project by executing the command inside the repository's root folder. 

npm run run-example

The example project does not use a pulled version of the framework. Instead it uses a locally build version of the framework. The mentioned command triggers a new build of the framework and starts the example project using this build. However, changes you make to the framework itself won't have any effect since the build is just executed once. If you want to play with the framework and see the consequences of your changes in the example project this page is of interest for you.

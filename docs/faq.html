Problems using the SVG format as React Component with Vite

In a few scenarios, the framework requires the use of SVG Files (e.g., Icons). To be able to set the color of the SVG inside the Framework, the Icons must be imported as React Components.

Use the following steps to solve issues regarding the SVG import as React Components:

Install the vite-plugin-svgr with the command "npm i vite-plugin-svgr" and configure it as shown here. If this doesn’t help, further possible solutions can be found here.

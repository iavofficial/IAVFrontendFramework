FAQ

As mentioned in the beginning of this documentation, bug reports and support requests are created using GitLab Issues or by writing an E-Mail to the official support address. This page lists common questions and problems and links corresponding issues.

General problems when downloading npm packages or the framework

If you have any problems with downloading the framework or any other npm package you should check your .npmrc file which should have the following 

registry=https://artifactory.iav.com/artifactory/api/npm/npm/

Problems using the SVG format as React Component with Vite

In a few senarios the framework require the use of SVG Files (e.g. Icons). To be able to set the color of the SVG inside the Framework the Icons must be importet as React Components.

Use the following steps to be able to solve issues regarding the SVG import as React Components:

Install the vite-plugin-svgr with the command "npm i vite-plugin-svgr" and configure it the way shown here. If this doesnt help further possible solutions can be found here.
